## Содержимое репозитория

- `homework_modules.ipynb` — основной ноутбук с реализацией модулей, функций активации, функций потерь, оптимизаторов и моделей.
- `homework_test_modules.ipynb` — ноутбук с тестами для проверки корректности реализованных классов и методов.

## Что реализовано в работе

В рамках задания вручную реализованы основные элементы, используемые при построении нейронных сетей:

- полносвязные слои (`Linear`);
- свёрточные слои (`Conv2d`);
- pooling-слои (`MaxPool2d`, `AvgPool2d`);
- слой `Flatten`;
- `Dropout`;
- `BatchNormalization`;
- функции активации:
  - `ReLU`
  - `LeakyReLU`
  - `ELU`
  - `SoftPlus`
  - `GELU`
- функции потерь:
  - `ClassNLLCriterion`
  - `ClassNLLCriterionUnstable`
  - `MSECriterion`
- оптимизаторы:
  - `SGD`
  - `Adam`

## Реализованные модели

На основе созданных модулей были собраны и обучены несколько моделей:

1. **Модель для задачи регрессии**  
   Используется полносвязная нейронная сеть для решения задачи мультирегрессии.

2. **Модель для задачи классификации MNIST**  
   Реализована сверточная нейронная сеть для распознавания рукописных цифр.

3. **Автоэнкодер**  
   Построена модель автоэнкодера для восстановления изображений.
